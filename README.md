## 📚 Description
In this repository, I have created a game that plays a background sound when a user clicks on a button. The game features English alphabets, and each button plays the corresponding alphabet sound.

## 🎊 Features
* Interactive buttons for each English alphabet
* Plays the sound of the corresponding alphabet when a button is clicked
* Simple and intuitive design

## 🛠️ Tech Stack
* HTML: Used for structuring the content of the game
* CSS: Used for styling the game

## 🗂️ Project Structure
The project consists of the following files:
* `README.md`: This file, containing information about the project
* `index.html`: The main HTML file for the game
* `script.js`: The JavaScript file containing the game logic
* `style.css`: The CSS file for styling the game
* `a1.mp3` to `z.mp3`: Audio files for each English alphabet

## 📦 Installation
To run the project, simply clone the repository and open the `index.html` file in a web browser.

## 🎮 Usage
1. Open the `index.html` file in a web browser
2. Click on a button to play the corresponding alphabet sound
